# Omni-Scribe

Omni-Scribe is an AI-powered documentation generator for [Omnigraph](https://github.com/ModernRelay/omnigraph) pull requests (PRs). It automatically fetches a PR from GitHub, processes the code diffs and metadata using Gemini, and generates high-quality changelogs and technical explainers.

---

## Prerequisites

- **Go**: Version 1.22 or higher.
- **Gemini API Key**: You need a Gemini API key. Get one from Google AI Studio.
- **GCP Service Account Key (Optional)**: Only required if you want to store documentation in Google Cloud Storage (GCS) rather than locally.

---

## Local Usage (Terminal)

### 1. Installation

Install the CLI directly using Go:

```bash
go install github.com/Taiwrash/omni-scribe@latest
```

*(Alternatively, run `go build` in the repository root to create a local binary).*

### 2. Configure Environment Variables

Set your Gemini API Key in your shell:

```bash
export GEMINI_API_KEY="your_gemini_api_key_here"
```

If you plan to store documentation in Google Cloud Storage (GCS) instead of the local filesystem, also set:

```bash
export GCS_BUCKET="your-gcs-bucket-name"
export GOOGLE_APPLICATION_CREDENTIALS="/path/to/your/gcp-service-account-key.json"
```

### 3. Generate Docs for a PR

To generate documentation for a specific Omnigraph PR (e.g., PR `#42`):

```bash
omni-scribe generate --pr 42
```

By default, this will save the output JSON to `./data/prs/42.json` (or to your GCS bucket if `GCS_BUCKET` is configured).

### 4. Serve the Docs

Start the local documentation web server to view the generated explainers and changelogs:

```bash
omni-scribe serve --port 8080
```

Open your browser and navigate to `http://localhost:8080` (or `http://localhost:8080/pr/42` to view a specific PR).

## CI/CD Automation (GitHub Actions)

You can run Omni-Scribe automatically on pull request merges.

### 1. Create a Workflow File
Add a new workflow file at `.github/workflows/omni-scribe.yml` with the following configuration:

```yaml
name: Generate Omni-Scribe Docs

on:
  pull_request:
    types: [closed]

jobs:
  generate-docs:
    # Only run when a PR is merged (not just closed)
    if: github.event.pull_request.merged == true
    runs-on: ubuntu-latest
    env:
      GCP_SA_KEY: ${{ secrets.GCP_SA_KEY }}

    steps:
      - name: Set up Go
        uses: actions/setup-go@v5
        with:
          go-version: "1.22"

      - name: Install omni-scribe
        run: go install github.com/Taiwrash/omni-scribe@latest

      - name: Authenticate to Google Cloud
        if: ${{ env.GCP_SA_KEY != '' }}
        uses: google-github-actions/auth@v2
        with:
          credentials_json: ${{ env.GCP_SA_KEY }}

      - name: Generate and Save PR Docs
        env:
          GEMINI_API_KEY: ${{ secrets.GEMINI_API_KEY }}
          GCS_BUCKET: ${{ vars.GCS_BUCKET || 'omni-scribe-docs' }}
        run: omni-scribe generate --pr ${{ github.event.pull_request.number }}

      - name: Post PR comment
        if: success()
        uses: actions/github-script@v7
        with:
          script: |
            const prNumber = context.payload.pull_request.number;
            const baseUrl = '${{ vars.OMNI_SCRIBE_URL || 'https://omni-scribe-docs.run.app' }}';
            const docsUrl = `${baseUrl}/pr/${prNumber}`;
            await github.rest.issues.createComment({
              owner: context.repo.owner,
              repo: context.repo.repo,
              issue_number: prNumber,
              body: `📖 **Docs generated by Omni-Scribe**\n\nView the auto-generated documentation for this PR:\n🔗 ${docsUrl}\n\n_Powered by [omni-scribe](https://github.com/Taiwrash/omni-scribe) + Gemini_`
            });
```

### 2. Configure GitHub Secrets and Variables
In your GitHub repository, navigate to **Settings** > **Secrets and variables** > **Actions** and add the following:

#### Repository Secrets
* **`GEMINI_API_KEY`** (Required): Your Google Gemini API Key.
* **`GCP_SA_KEY`** (Optional): The JSON content of your Google Cloud Service Account key (used to upload to GCS).

#### Repository Variables
* **`GCS_BUCKET`** (Optional): The name of your GCS bucket (defaults to `omni-scribe-docs`).
* **`OMNI_SCRIBE_URL`** (Optional): The base URL of your deployed Omni-Scribe web server (defaults to `https://omni-scribe-docs.run.app`).
